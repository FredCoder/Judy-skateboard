Judy-skateboard
======
**Judy-skateboard** 是一个DIY电动滑板的程序集合,里面包含了制作过程中的实验代码,Arduino的控制代码,以及Android APP的遥控程序.


## 兼容
 * [Android Apps](http://www.android.com)
 * [Arduino UNO](http://www.arduino.cc)
